1. The Dataset

We'll use this sample student data throughout the tutorial. Every function below builds on it.

const students = [
  { name: "Alice",  score: 92, grade: "A" },
  { name: "Bob",    score: 78, grade: "B" },
  { name: "Carol",  score: 85, grade: "B" },
  { name: "Dan",    score: 64, grade: "D" },
  { name: "Eve",    score: 91, grade: "A" },
  { name: "Frank",  score: 73, grade: "C" },
  { name: "Grace",  score: 88, grade: "B" },
  { name: "Hank",   score: 55, grade: "F" },
  { name: "Iris",   score: 97, grade: "A" },
  { name: "Jack",   score: 70, grade: "C" },

Rendered data

2. Extracting Columns — unpack

unpack pulls a single column out of an array of objects — the foundation for everything that follows.

const scores = u.unpack(students, "score");
// → [92, 78, 85, 64, 91, 73, 88, 55, 97, 70]
Result

3. Summary Statistics

Compute common descriptive stats in one call each.

u.mean(scores)          u.median(scores)
u.mode(scores)          u.stdDev(scores)
u.min(scores)           u.max(scores)
u.sum(scores)           u.extent(scores)
u.percentile(scores, 25)  u.percentile(scores, 75)
Results

4. Sorting & Filtering

Sort objects by any key. sortBy returns a new array — the original is never mutated.

u.sortBy(students, "score", "desc")
Sorted by score (descending)

Filter with a predicate function.

u.filterRows(students, r => r.score >= 80)
Students scoring 80+

Extract unique values and frequency counts.

u.unique(u.unpack(students, "grade"))
u.counts(u.unpack(students, "grade"))
Unique grades & counts

5. Grouping & Aggregation — groupBy

Group rows by a column, then aggregate each group — a common pattern in data analysis.

const byGrade = u.groupBy(students, "grade");
// Then compute count, mean score per group
Grouped summary

6. Binning & Visualization — bin, colorScale

Create histogram bins, then color them with an interpolated color scale.

const bins = u.bin(scores, 5);
const heat = u.colorScale(u.extent(scores), ["#4a90d9", "#f5a623", "#d0021b"]);
Histogram

7. Scales — linearScale, normalize

linearScale maps values from one range to another — for example, scores to pixel widths.

const toWidth = u.linearScale([0, 100], [0, 500]);
// toWidth(92) → 460
Scores as horizontal bars (mapped to 0–500px)

normalize scales an array to [0, 1] (or any custom range).

u.normalize(scores)
// → values between 0 and 1
Normalized scores

8. Rolling Functions — rollingAverage, rollingSum

Compute moving averages or sums over a sliding window. Useful for smoothing noisy data.

u.rollingAverage(scores, 3)
u.rollingSum(scores, 3)
Original vs. 3-point rolling average & sum

9. Formatting — formatNumber, formatPercent

Clean display of numbers and percentages for labels and annotations.

u.formatNumber(1234.5)      // → "1,234.50"
u.formatNumber(79.3, 1)     // → "79.3"
u.formatPercent(0.793)       // → "79.30%"
u.formatPercent(0.793, 0)    // → "79%"
Formatted output

10. Parsing Helpers — autoType, parseDates

autoType converts string values (e.g. from CSV) to appropriate types — numbers, dates, or null.

const raw = [
  { name: "Alice", score: "92", enrolled: "2024-01-15" },
  { name: "Bob",   score: "78", enrolled: "2024-02-01" },
];
u.autoType(raw);
// score becomes a number, enrolled becomes a Date
autoType result

parseDates targets a specific column with an optional format string.

const rows = [
  { event: "Midterm", date: "15/03/2024" },
  { event: "Final",   date: "20/06/2024" },
];
u.parseDates(rows, "date", "DD/MM/YYYY");
parseDates result
